## https server on port

Here is a **starter** post with an example codeblock.


```bash
start () {
	if [[ $# -eq 0 ]]
	then
		echo LAN address: `ifconfig | grep -A 4 en0 | grep -w inet | awk -F' ' '{print $2}'`:9000
		python3 -m http.server 9000
	else
		echo LAN address: `ifconfig | grep -A 4 en0 | grep -w inet | awk -F' ' '{print $2}'`:$1
		python3 -m http.server $1
	fi
}
```
